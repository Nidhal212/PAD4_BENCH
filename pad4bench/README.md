# pad4bench python package

The main package implementing the curation pipeline, splits, benchmarking,
figures, and tables.

```
pad4bench/
├── __init__.py
├── curation/        # 13-stage curation pipeline
├── splits/          # 5 split protocol generators
├── benchmark/       # training and evaluation
├── figures/         # figure generators
├── tables/          # table generators
└── utils/           # shared helpers (logging, paths, seeds)
```

Each subpackage has its own README.
